# Dogs-PI-Vite

SPA proyecto individual para Henry. 
Para el servidor utilice JavaScript, SQL junto con ExpressJS y Sequelize.
Para el cliente las tecnologicas empleadas fueron JavaScript, React, Redux y Css.
